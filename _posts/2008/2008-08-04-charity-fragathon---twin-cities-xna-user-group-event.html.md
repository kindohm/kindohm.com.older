<p>The <a href="http://www.twincitiesxnausergroup.com/" target="_blank">Twin Cities XNA User Group</a> is putting on a charity event on 9/24/2008 entitled the <a href="http://charityfragathon.com/" target="_blank">Charity Fragathon</a>:</p>
  
<p><a title="http://charityfragathon.com/" href="http://charityfragathon.com/">http://charityfragathon.com/</a></p>

As long as some of the games are played on the <a href="http://www.bungie.net/forums/posts.aspx?h3fileid=45109209" target="_blank">Alpaca Death Map</a> then I'll have a chance :)</p>
