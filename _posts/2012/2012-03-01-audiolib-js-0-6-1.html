--- layout: post title: "audiolib.js v0.6.1" ---

The powerful JavaScript audio API audiolib.js was recently released as version v0.6.1. You can download the fully built library at github.com/jussi-kalliokoski/audiolib.js/downloads.

Many congrats go to Jussi Kalliokoski, who has done the bulk of the work on this project.

I'm especially excited about this release because I actually contributed a small amount to it. I haven't been active in useful open source projects in the past, so actually seeing a release that includes code I worked on is very satisfying.

If you are a synthesizer geek and a programmer, I highly recommend checking out audiolib.js. It's a big sandbox of fun!
